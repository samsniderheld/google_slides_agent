How to Use

Authentication Steps:

  1. Login: Click the "Login with Google" button
  2. Grant Permissions: Allow access to Google Slides and Drive
  3. Automatic Redirect: You'll be redirected back to the app automatically
  4. Ready: Start creating presentations!

Creating Presentations:

  1. Authenticate: Complete Google authentication first
  2. Enter Concept: Describe your presentation idea in detail
  3. Add Title: Give your presentation a name
  4. Choose Template (optional): Use default or specify a Google Slides template ID
  5. Generate: Click the button and wait for your presentation to be created
  6. Open: Click the generated URL to view and edit your presentation

Tips:

  • Be specific about your target audience and goals
  • Include key points you want to cover
  • Mention the tone (professional, creative, casual, etc.)
  • Make sure you have edit access to the source template

Google Cloud Console Setup:

  1. Create OAuth 2.0 credentials for "Web application"
  2. Add authorized redirect URI: http://localhost:5000/oauth/callback
  3. Enable Google Slides API and Google Drive API
  4. Copy Client ID and Client Secret to your .env file
